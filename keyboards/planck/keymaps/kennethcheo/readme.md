# Planck Layout

## Current Features
1. Tap dance L-shift and caps lock
2. Triple tap dance brackets (,[,{
3. Hold: Raise, lower; Press: Enter, backspace
4. Ctrl and Alt keys moved one key right
5. Hold F for ctrl
